# Model Card: ViT Robustness and XAI

## Dataset
Target evaluation data is an ImageNet-mini style clean subset plus generated ImageNet-C-style corruptions using `imagecorruptions`.

## Model
Main model: pretrained `vit_b_16`. Baseline: pretrained `resnet50`.

## Evaluation
Primary metric: mean Corruption Error (mCE) computed by `scripts/compute_mce.py`. Secondary metric: clean and corrupted top-1 accuracy.

## Explainability
Supported methods: attention rollout and GradCAM. Heatmaps are generated by `scripts/xai/generate_heatmaps.py`.

## Limitations
This repository does not include ImageNet images or model weights beyond public pretrained torchvision weights.

## Ethical Considerations
Robustness and saliency explanations should not be treated as full causal explanations. Report failure cases alongside positive examples.
